JSLintMate

Quick, simple JSLint (or JSHint) in TextMate. Hurt your feelings in style.

Fork me on GitHub

Installation

Download JSLintMate 1.0.4

Download JSLintMate.tmbundle and double-click it. TextMate should install it for you automatically—that’s all. Everything works offline.

Git lovers: You can also install via Git. Check the readme for details.

Usage

  1. Open a JS file in TextMate.
  2. Hit control-L to run it through JSLint (or control-shift-L for JSHint).
  3. Fix and repeat.

Wait, why do I need this?

JSLintMate makes it quick and easy to run your code through either JSLint (Douglas Crockford's powerful JS code quality tool) or JSHint (a community-driven project, based on JSLint, that’s kinder to your feelings). No need to copy-paste JS into your browser—JSLintMate runs right within TextMate.

If you’re new to this, or have plenty else to do, the simple, shiny UI just lets you get things done.

Experts: You can put JSLint/JSHint options in a config file, either globally or per project. Add the config file to a shared repository, and it’s perfect for maintaining standards in team development. More info is in the readme.
